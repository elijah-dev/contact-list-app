Приложение создано с помощью `create-react-app` поэтому сначала нужно само собой сделать

```bash
npm install
```

и потом

```bash
npm start
```

Логин и пароль для авторизации уже введены в форму

Я использую `json-placeholder` в качестве API, и он не сохраняет изменения между заросами,
поэтому при попытке удаления и изменения вновь созданных контактов всегда будет возвращатся 404

В качестве UI фреймворка использую `Material UI`
